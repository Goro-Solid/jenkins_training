### Tworzenie folderu

Aby stworzyć nowy folder, otwórz Jenkinsa http://localhost:8080/, następnie w glównym menu 
    
    New Item ->
        -> Wprowadź nazwę 002
        -> Zaznacz 'folder' 
        -> Kliknij OK
            -> Kliknij Save

Wszystkie moduly i cwiczenie w tym szkoleniu bedziemy przechowywać w folderach z numerem modulu np. 002,003,etc ..
